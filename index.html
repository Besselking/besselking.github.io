This html page is styled like a latex source file.

This “project” is inspired by html wysiwyg. All latex commands are added with psuedo elements (::before, :after) and do not exist in the html source file. Mixed with a little bit of normal styling to make it a bit more pleasant to look at.

Elements that dont work

  • replacement elements like img, or br

  • h6, there is no subsubparagraph in the default article class
